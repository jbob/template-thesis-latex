# Easy, minimal template for a (german) thesis/dissertation

## Compile

    $ pdflatex template
    $ bibtex template
    $ pdflatex template
    $ pdflatex template

## Dependencies

* LaTeX (and Bibtex) installation
* Koma-Script document classes
* libertine font package
* listings package
* microtype package
* hyperref package
* babel package
* booktabs package
* graphix package
* blindtext package
* acronym package
* setspace package
